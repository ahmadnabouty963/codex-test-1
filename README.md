# Nourish – Ernährungs-Tracker

Eine umfangreiche, responsive Ernährungs-Tracker-App mit reinem HTML, CSS und JavaScript.

## Funktionen

- Lebensmittel als gegessen oder geplant erfassen
- Kalorien, Protein, Kohlenhydrate, Fett, Zucker, Ballaststoffe und Salz berechnen
- Mahlzeiten nach Frühstück, Mittagessen, Abendessen und Snacks gruppieren
- Individuelle Tagesziele und Wasser-Tracking
- Wochenübersicht und Tracking-Serie
- Deutsch, Englisch und Arabisch mit RTL-Unterstützung
- Dark Mode und Light Mode
- Lokale Datenspeicherung im Browser ohne Login

## Starten

`index.html` im Browser öffnen oder einen lokalen Server starten, zum Beispiel:

```bash
npx serve .
```
